# Icons
Almost all icons come from <a target="_blank" href="https://material.io/resources/icons/?style=baseline">material icons</a> unless cart icon. Icons are located in a module `src/components/Icons.js` file.

`<{Icon name} className="icon [icon color class name] [icon size class name]"/>`
