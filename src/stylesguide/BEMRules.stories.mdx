# BEM(Block, Element, Modifier)

BEM (Block, Element, Modifier) is a component-based approach to web development. The idea behind it is to divide the user interface into independent blocks. 
This makes interface development easy and fast even with a complex UI, and it allows reuse of existing code without copying and pasting.

###
## Block

The component is unit of every block. If component is composed of other component, that only class name of native DOM base on current component block. Then components as an independent block for introduced.
