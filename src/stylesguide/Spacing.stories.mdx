# Spacing

### 
### 
## Padding Spacing

### top, right, bottom and left are 4px padding spacing.
```js live=true
padding-smaller
```

### top, and bottom are 4px padding spacing.
```js live=true
padding-top-bottom-smaller
```

### left, and right are 4px padding spacing.
```js live=true
padding-left-right-smaller
```

### top, right, bottom and left are 8px padding spacing.
```js live=true

```

### top, and bottom are 8px padding spacing.
```js live=true

```

### left, and right are 8px padding spacing.
```js live=true

```

### top, right, bottom and left are 16px padding spacing.
```js live=true

```

### top, and bottom are 16px padding spacing.
```js live=true

```

### left, and right are 16px padding spacing.
```js live=true

```
