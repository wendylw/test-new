# Layout

Almost all pages contain header, footer and container section. The outermost layer is `<main></main>` of every project.

## Layout CSS
Beep includes `absolute` `sticky` and `fixed` positioning methods.

```js live=true
.absolute-wrapper {
  position: absolute;
  left: 0;
  right: 0;
}

.sticky-wrapper {
  position: -webkit-sticky;
  position: sticky;
}

.fixed-wrapper {
  position: fixed;
  left: 0;
  right: 0;
  height: 100%;

  &__main {
    position: relative;
    min-height: 100vh;
  }

  &__container {
    width: 100%;
    height: 100%;
    overflow-x: hidden;
    overflow-y: auto;
  }
}
```
